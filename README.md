# Affiliate Payout System

A production-inspired backend system for processing affiliate commissions using batch payouts, asynchronous workers, MongoDB transactions, Redis, and BullMQ.

---

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Redis
- BullMQ

---

## Features

- User & Wallet Management
- Sale Management
- Advance Commission Processing
- Final Commission Reconciliation
- Withdrawal Processing
- Immutable Transaction Ledger
- Batch Processing
- MongoDB Transactions
